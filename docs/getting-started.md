# Getting Started with PDD

## Install via npx

```bash
npx @pcoliveira90/pdd init
```

## What it creates

```
.pdd/
  templates/
  commands/
  memory/
```

## First workflow

1. Describe the issue
2. Run recon
3. Create delta spec
4. Plan patch
5. Apply change
6. Verify

## Philosophy

PDD is about:
- safe changes
- understanding before coding
- minimizing risk
