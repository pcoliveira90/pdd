# PDD vs GSD vs Spec Kit

## Overview

| Framework | Focus | Strength |
|----------|------|---------|
| GSD | Execution speed | Fast iteration |
| Spec Kit | Specification | Structured planning |
| PDD | Existing systems | Safe changes |

---

## GSD
- Focus: do things fast
- Great for: quick builds
- Weakness: lacks deep safety model for legacy systems

## Spec Kit
- Focus: spec-driven development
- Great for: structured feature building
- Weakness: less focused on bugfix and real system evolution

## PDD
- Focus: change in existing systems
- Great for:
  - bugfix
  - incremental feature
  - production safety

---

## Key Differentiator

PDD is built for reality.

Not:
- ideal systems
- greenfield

But:
- legacy code
- production bugs
- evolving systems
